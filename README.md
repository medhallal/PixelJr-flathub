# PixelJr for Flathub

![PixelJr Icon](com.almaarijstudios.pxjr.png)

## About

**PixelJr** (Pixel Junior) is an educational desktop application designed to accompany the "Pixel Informatique et Ingénierie" textbook.

It offers a wide range of interactive activities, educational resources, quizzes, and mini-games to help learners strengthen their computer science skills in a fun and progressive way.

- **Interactive activities** – Hands-on exercises that promote learning through practice.
- **Educational resources** – Complementary content to deepen concepts covered in class.
- **Fun quizzes** – Interactive and engaging quizzes to evaluate knowledge.
- **Educational mini‑games** – Games designed to consolidate technical skills while playing.

The application integrates naturally with the *Pixel Informatique et Ingénierie* textbook, extending classroom lessons into a guided digital environment.

## Key Features

- Progressive learning path tailored to the textbook
- Intuitive, student‑friendly interface
- Tools that help teachers track progress and adapt activities
- Stimulating challenges to reinforce understanding of key computing concepts
